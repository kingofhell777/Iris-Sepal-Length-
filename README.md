# Iris Sepal Length Distribution Analysis

This project is a small Python notebook that analyzes the *Sepal Length* feature from the classic IRIS DATASET and compares it across the three Iris species.

---

WHAT THIS NOTEBOOK DOES :-

- Loads the Iris dataset from a CSV file
- Groups the data by species and calculates:
  - Mean
  - Median
  - Standard deviation
  - Minimum
  - Maximum
- Calculates overall central tendency values (mean, median, mode)
- Prints a recommendation on which measure (mean/median) should be reported
- Mentions limitations of the analysis

---

TECHNOLOGIES USED
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

---

DATASET USED
The notebook uses the Iris dataset stored as a CSV file.
